TaskWork v2 · 任务即对话

定时任务的报告会话不再进 Code 左栏（不再建「定时任务」分组）；任务在 TaskWork 页内自成闭环，对齐 Cowork 心智。

中栏 = 选中任务的会话流

  • 默认显示最新一次运行的完整对话（指令 → 工具足迹 → 报告），复用 Design 对话列的轻量渲染。
  • 正在运行时中栏 live 流式跟进（打字机同款），跑完自动定格。
  • 底部可就报告续聊（同一会话 resume，续聊轮不受只读限制、权限照常走全局卡）。

    • 右栏 = Runs + 配置

  • 点任一 run 切换中栏回放；✓/✕/● 状态与 tokens 一眼可见。
  • 配置从中栏移到右栏摘要卡，「编辑配置」展开完整表单（新建任务时中栏即表单，同现版）。

    • 数据层改动 迁移

  • cron 会话 init 不再归组进 Code 栏，改标记 hidden（Code 会话列表过滤掉）。
  • 存量「定时任务」分组的会话一次性迁移为 hidden，分组消失。
  • Recents / Runs 点击均在页内打开，不再跳回 Code。
